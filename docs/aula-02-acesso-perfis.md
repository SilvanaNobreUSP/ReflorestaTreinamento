# Aula 02 - Acesso e perfis de usuário

## Objetivo da aula

Explicar como acessar a plataforma, reconhecer o perfil de usuário e entender as permissões disponíveis para cada tipo de atuação.

## Explicação principal

O acesso à ReflorestaSP deve seguir os procedimentos internos de autenticação e autorização. Cada usuário pode ter permissões diferentes, de acordo com sua função, unidade, projeto ou responsabilidade operacional. Antes de executar qualquer alteração, confirme se o perfil ativo corresponde à atividade pretendida.

![Ilustração de acesso e perfis](images/aula-02-acesso-perfis.svg)

## Passo a passo

1. Abra o endereço da plataforma no navegador.
2. Informe suas credenciais conforme o padrão definido pela SEMIL.
3. Confirme se o nome do usuário aparece corretamente no ambiente.
4. Acesse a área de perfil ou configurações da conta.
5. Verifique as permissões disponíveis para consulta, cadastro, edição e exportação.
6. Caso alguma permissão esteja incorreta, registre a necessidade de ajuste pelo canal interno adequado.
