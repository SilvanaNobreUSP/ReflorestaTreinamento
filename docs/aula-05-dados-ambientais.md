# Aula 05 - Dados ambientais e territoriais

## Objetivo da aula

Explicar como consultar e interpretar informações ambientais e territoriais relacionadas às áreas cadastradas.

## Explicação principal

Os dados ambientais e territoriais ajudam a contextualizar cada área e apoiar decisões técnicas. Eles podem incluir localização, recortes administrativos, camadas de referência, restrições, prioridades e informações úteis ao planejamento da restauração.

![Ilustração de dados ambientais](images/aula-05-dados-ambientais.svg)

## Passo a passo

1. Acesse uma área já cadastrada.
2. Localize a seção de dados ambientais ou territoriais.
3. Verifique as informações disponíveis para a área selecionada.
4. Consulte mapas, camadas ou tabelas associadas, quando houver.
5. Compare as informações exibidas com documentos ou bases oficiais disponíveis.
6. Registre dúvidas ou inconsistências para validação técnica.
